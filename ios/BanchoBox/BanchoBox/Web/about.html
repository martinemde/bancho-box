About Bancho Box

Bancho Box is a comprehensive database application for Dave the Diver, built with modern web technologies to help players navigate the game's complex culinary system.

Features

  • Complete dish database with unlock conditions, pricing, and party bonuses
  • Comprehensive ingredient catalog with sources and gathering information
  • Party event listings with associated dishes and bonus multipliers
  • Dark/light theme support for comfortable viewing
  • Responsive design for desktop and mobile devices

Technology Stack

This application is built with SvelteKit and styled with Skeleton UI for a modern, accessible user experience.
